## BackEnd da Imersão Fullcycle 14 - Sistema de rastreamento de veículos
#### Backend project made with nestJs using the google maps API
  
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
